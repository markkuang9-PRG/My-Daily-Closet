# GitHub Discussion Post

## Suggested title

Looking for early contributors: React + Firebase + AI wardrobe app

## Post body

I am building **My Daily Closet**, an open-source AI wardrobe app built with React, Firebase, and Gemini.

The current app already supports:

- clothing photo upload with AI tagging
- weather-aware outfit recommendations
- 90-day idle item detection
- AI-generated resale copy

Right now I am focused on turning the prototype into a cleaner MVP.

The most useful contribution areas at the moment are:

- React cleanup and component polish
- Firebase reliability and error handling
- AI prompt quality and response validation
- performance improvements
- accessibility and mobile UX polish

If you want to contribute, start here:

- [CONTRIBUTOR_START_HERE.md](../../CONTRIBUTOR_START_HERE.md)
- [GOOD_FIRST_TASKS.md](../../GOOD_FIRST_TASKS.md)

The first public issues are intended to stay narrow and reviewable. I am trying to keep scope tight so new contributors can get merged quickly without getting pulled into a broad rewrite.

If you are interested, reply here or pick one small issue once the first wave is posted.
